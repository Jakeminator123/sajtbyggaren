# MIN_IDE templates (referens)

Den här mappen innehåller en kopia av operatörens `MIN_IDE/`-material för spårbar referens i repot.

Allt innehåll här är inspirations- och jämförelsematerial. Det är inte produktkod och används inte av runtime eller backoffice.

## Struktur

- `dossiers/` - exempel på dossier-format, capability-map och implementationsunderlag
- `scaffolds/` - exempel på scaffold-manifest och filstrukturer
- `scaffold-variants/` - exempel på variant-filer och embeddings-index
- `prompts/` - exempel på init/follow-up promptmaterial

Källan för kopian är `MIN_IDE/` i workspace-roten. Originalet kan fortsätta ändras lokalt utan att påverka historiken i `referens/`.
