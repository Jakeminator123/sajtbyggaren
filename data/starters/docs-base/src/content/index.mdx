# Overview

This starter is a neutral documentation base for education and course
experiences. It keeps the Nextra plugin, markdown/MDX authoring, the
`/docs/[[...mdxPath]]` catch-all route, hybrid Nextra + Pagefind search,
and a light/dark theme toggle.

The generated scaffold can replace this placeholder content with program
pages, lecture lists, and resource libraries.

## What belongs here

- reusable course documentation structure
- markdown and MDX content in `src/content/`
- a manually maintained sidebar in `src/app/layout.tsx` that scaffolds
  must keep in sync with new content (see Authoring)
- neutral layout and theme defaults

## What stays outside

- authentication
- database access
- payments
- analytics
- customer specific copy
