# Overview

This starter is a neutral documentation base for education and course
experiences. It keeps the Nextra docs theme, markdown authoring, sidebar
navigation, search, and light or dark mode.

The generated scaffold can replace this placeholder content with program
pages, lecture lists, and resource libraries.

## What belongs here

- reusable course documentation structure
- markdown and MDX content
- sidebar navigation metadata
- neutral layout defaults

## What stays outside

- authentication
- database access
- payments
- analytics
- customer specific copy
