# Authoring

Nextra renders markdown and MDX files from the content directory. Use `_meta`
files to control sidebar labels and ordering.

## Markdown support

The starter keeps standard Nextra docs behavior:

- headings generate table of contents entries
- sidebar items follow content metadata
- code blocks are rendered by the theme
- search uses the docs theme search integration

## Scaffold integration

Later scaffolds can replace or extend the content directory without changing the
theme setup.
